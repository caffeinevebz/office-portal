# Original Ledgify artwork

Drop the original logo PNGs in **either the `public/` root or this
`public/brand/` folder** (exact names below) and the app picks them up
automatically — no code changes needed:

| File | Used for | Recommended size |
| --- | --- | --- |
| `logo-full.png` | The login screen's logo panel (web + mobile) | the full square artwork, ≥ 1024×1024 |
| `icon-512.png` | Phone home-screen / PWA icon | 512×512, square, full-bleed |
| `icon-192.png` | Browser tab favicon + Apple touch icon | 192×192, square, full-bleed |

Until a file exists, the app falls back to its built-in recreation of the
logo, so uploading any subset is fine. After adding the icons, phones that
already installed the app must remove and re-add it to refresh the icon.
